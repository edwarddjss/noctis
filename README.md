# Noctis

Adaptive night vision for gaming. Automatically adjusts gamma based on on-screen brightness.

## Features

- **Smart Auto-Adjustment** - Real-time shadow lifting for dark scenes
- **Multi-Monitor Support** - Target specific displays
- **Global Hotkey** - Toggle with a single keypress
- **Minimal UI** - Clean, unobtrusive interface
- **System Tray** - Runs quietly in the background

## Install

Download `Noctis.exe` from [Releases](https://github.com/edwarddjss/noctis/releases).

## Build from Source

```bash
cd app
npm install
npm run tauri build
```
